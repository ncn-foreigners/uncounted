# Summary of Bootstrap Population Size Results

Prints the same table as
[`print.uncounted_boot`](https://ncn-foreigners.github.io/uncounted/reference/print.uncounted_boot.md),
followed by a per-group summary of the bootstrap distribution: mean,
standard deviation, 2.5\\ assessing the shape of the bootstrap
distribution (e.g., skewness) and comparing the spread across groups.

## Usage

``` r
# S3 method for class 'uncounted_boot'
summary(object, ...)
```

## Arguments

- object:

  An `"uncounted_boot"` object.

- ...:

  Additional arguments (ignored).
